# סימולטור מערכת שעות חכם 📅

כלי אינטראקטיבי חכם המאפשר לסטודנטים לתכנן, לבנות ולייצא את מערכת השעות האקדמית שלהם בקלות, ללא צורך בניירות וטיוטות. המערכת רצה ישירות בדפדפן - אין צורך בהתקנת תוכנה!

🚀 **[למעבר לסימולטור לחצו כאן](https://danielstern98.github.io/Schedule-Simulator/)**

---

## ✨ פיצ'רים מרכזיים
* **תצוגה ויזואלית חכמה:** שני יומנים מקבילים - מאגר האפשרויות והמערכת האישית שלך.
* **זיהוי התנגשויות אוטומטי:** המערכת תתריע באמצעות רעידה ומסגרת אדומה במקרה של חפיפת זמנים.
* **מנגנון חוקים ובחירות:** תמיכה בקבוצות בחירה (למשל: בחירת קורס רוח אחד מתוך כמה) וקבוצות מפגשים חובה (למשל: הרצאה שגוררת אחריה תרגול ספציפי).
* **סנכרון גלילה וקו זמן (לייזר):** כלים נוחים להשוואת זמנים בין שני הלוחות.
* **יצוא המערכת:** שמירת המערכת שבנית כתמונה (PNG) או הורדת רשימת קורסים קריאה כקובץ טקסט.

---

## 🛠️ איך משתמשים?
1. היכנסו לקישור של הסימולטור.
2. לחצו על כפתור **"הורד קובץ אקסל מוכן למילוי"**.
3. מלאו את האקסל עם נתוני הקורסים שלכם לפי ההנחיות (פירוט למטה).
4. העלו את הקובץ המוכן חזרה לסימולטור.
5. התחילו לשבץ! לחיצה על קורס במאגר תעביר אותו למערכת שלכם.

---

## 📝 מדריך למילוי קובץ האקסל
קובץ האקסל חייב להכיל את העמודות הבאות בדיוק לפי הסדר הזה:

### עמודות חובה:
* **(A) שם קורס:** שם הקורס (לדוגמה: *פיזיקה ג*).
* **(B) סוג ה/ת/מ:** סוג השיעור - ניתן לכתוב *ה/ת/מ* או *הרצאה/תרגול/מעבדה*.
* **(C) יום:** אות בודדת מ-א' עד ו' (לדוגמה: *ב*).
* **(D) שעת התחלה:** מספר רציף של 3-4 ספרות בפורמט 24 שעות (לדוגמה: *815* עבור 08:15 בבוקר, או *1630* עבור 16:30 אחה"צ).
* **(E) שעת סיום:** מספר רציף באותו הפורמט.
* **(F) מרצה:** שם המרצה.

### עמודות רשות מתקדמות:
* **(G) בחירת קורס מתוך כמה קורסים:** 
  אם יש לכם אשכול קורסים שמתוכו חובה לבחור רק אחד (למשל קורסי רוח), רשמו תו או מילה זהה (למשל: "רוח") בכל השורות של קורסים אלו. הסימולטור יתריע אם תבחרו יותר מאחד.
* **(H) בחירת קבוצת מפגשים:** 
  אם קורס דורש שילוב של הרצאה + תרגול ספציפי, תנו לשניהם את אותו התו בעמודה זו. לחיצה על אחד מהם תשבץ אוטומטית גם את השני.
